# Diamond Education System - Quick Start Guide

## 🚀 Get Started in 60 Seconds

### Step 1: Install Dependencies
```bash
npm install
```

### Step 2: Run Development Server
```bash
npm run dev
```

### Step 3: Open in Browser
```
http://localhost:3000
```

## 🧪 Test Each Role (2 Minutes Per Role)

### Admin Dashboard
1. Go to login page → click "Sign In"
2. Enter: `admin@diamond.edu` / `admin123`
3. Explore:
   - Dashboard → view system metrics
   - Users → create, edit, delete users
   - Analytics → filter and export data
   - Groups → manage courses
   - Settings → configure system
   - AI Generator → create content

### Student Dashboard  
1. Login with: `student@diamond.edu` / `student123`
2. Explore:
   - Dashboard → view D'Coins balance
   - Lessons → play lessons
   - Vocabulary → learn words
   - Games → play learning games
   - Homework → submit assignments
   - Leaderboard → check rankings
3. **Test AI Chat**:
   - Click chat bubble (bottom right)
   - Send 3 messages (free)
   - Send 4th message → see "-5 D'Coins" warning

### Teacher Dashboard
1. Login with: `teacher@diamond.edu` / `teacher123`
2. Explore:
   - Dashboard → see classes
   - My Groups → create/manage groups
   - Attendance → mark students present/absent
   - Tests → create exams
   - Analytics → view student progress

### Support Dashboard
1. Login with: `support@diamond.edu` / `support123`
2. Explore:
   - Dashboard → view pending requests
   - Respond to requests (opens modal)
   - View today's schedule

## 🐛 Troubleshooting

### Chat Not Appearing?
- Refresh the page
- Check browser console for errors
- Ensure you're logged in (or on public page)

### Modal Won't Open?
- Check browser console
- Try clicking button again
- Refresh page if needed

### Search Not Working?
- Clear search box
- Try typing one word
- Check network tab for API calls

### Style Issues?
- Clear browser cache (Ctrl+Shift+Delete)
- Hard refresh (Ctrl+Shift+R)
- Try different browser
